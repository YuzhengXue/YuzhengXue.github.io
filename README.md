# Personal Academic Website

This is a zero-build static starter for a personal academic website. Open
`index.html` directly in a browser to preview it.

## What to edit first

- Replace `Your Name`, affiliation, location, and email in `index.html`.
- Update the hero paragraph and About section with your academic bio.
- Replace placeholder publications with real titles, venues, links, and years.
- Replace `YN` in the portrait placeholder with your initials, or later swap it
  for a real profile photo.

## Suggested next steps

1. Add your real academic information.
2. Add a `cv.pdf` file and link it from the header or hero.
3. Add a `publications.json` file so publication entries can be maintained as data.
4. Deploy the static site with GitHub Pages, Netlify, or Vercel.
